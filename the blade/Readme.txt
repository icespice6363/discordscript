Use the F8 key to open using the keyboard.
Use RB+Left to open using the gamepad.

Use the following keys to navigate:

Backspace - Back
Enter - Select
UpArrow - Up
DownArrow - Down
LeftArrow - Left
RightArrow - Right

Controller input:

B/Circle - Back
A/X - Select
DPAD Up - Up
DPAD Down - Down
DPAD Left - Left
DPAD Right - Right

You can change the keyboard and controller Blade open/close settings as well as other Blade defaults in the 'BladeConfig' file, which is found in the BladeStuff folder inside your GTA Directory. 




How to install:
Place Blade.asi and BladeStuff (and any other important-looking files) in the GTA V directory.
